# 📊 Mobile Sales Dashboard | Power BI Project

## 🔹 Project Overview
This project is an interactive Mobile Sales Dashboard created using Microsoft Power BI to analyze sales trends, customer behavior, and business KPIs.

## 🔹 Features
- Sales Performance Analysis
- KPI Tracking
- Interactive Filters & Slicers
- Payment Method Analysis
- Regional Sales Insights

## 🔹 Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## 🔹 Key Insights
- Total Sales: 769M+
- Total Transactions: 4K+
- Quantity Sold: 19K+
